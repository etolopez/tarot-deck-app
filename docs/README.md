# GitHub Pages Legal Documents

This directory contains the legal documents for Tarot Deck App, formatted for GitHub Pages.

## Setup Instructions

1. **Enable GitHub Pages**:
   - Go to your repository on GitHub
   - Click on "Settings"
   - Scroll down to "Pages" section
   - Under "Source", select "Deploy from a branch"
   - Choose branch: `main`
   - Choose folder: `/docs`
   - Click "Save"

2. **Access Your Pages**:
   - Your legal documents will be available at:
   - `https://etolopez.github.io/tarot-deck-app/`

3. **Update Documents**:
   - Edit the markdown files in the root directory
   - The HTML files in `/docs` will need to be regenerated
   - Or link directly to the markdown files on GitHub

## Files

- `index.html` - Landing page with links to all legal documents
- `style.css` - Styling for all pages
- `PRIVACY_POLICY.html` - Privacy Policy page
- `TERMS_OF_SERVICE.html` - Terms of Service page
- `LICENSE.html` - License page
- `COPYRIGHT.html` - Copyright page
